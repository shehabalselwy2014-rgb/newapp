# Eisenhower Matrix Pro

This workspace contains the Eisenhower Matrix productivity web app.

## Project structure

- `index.html` — main application page
- `css/` — application styles
- `js/` — application logic
- `assets/` — static image and icon assets
- `libs/` — third-party library bundles
- `data/` — static JSON content

## Notes

- Replace the placeholder library files in `libs/` with actual bundles when available.
- Add real image assets into `assets/images/` and `assets/themes/`.
- The app uses localStorage for user data and tasks.
